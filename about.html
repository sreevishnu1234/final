Description:

This project is a web-based tool designed to analyze log files generated by an orchestrator. It provides users with the ability to upload log files, analyze the content, and display relevant information in a user-friendly
manner. The key features and components of this project include:

1.User Interface:

- A clean and user-friendly web interface with a centered title.
- File input for users to upload log files.
- Buttons for analyzing log files and resetting the interface.
- A dropdown menu to select between two tables: "Job Creation and Operations" and "Extracted Field Values."

2.Search and Filter:

- A search bar allows users to search for specific content within the displayed table.
- Sorting options are available to arrange data in ascending or descending order based on job IDs.

3.Design:

- The project features a responsive and visually appealing design.
- Stylish CSS is applied for tables, buttons, and the search input field.
- The interface consists of colour scheme for easy readability.

Future Enhancements:

- Additional sorting options for other table columns.
- Exporting table data to different file formats.
- Improved error handling for invalid log files.
